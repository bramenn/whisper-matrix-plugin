## Run this project

Install the requerimets
- `pip install -r requirements.txt`

Set variables envs
- `vim .env`

```
MATRIX_USER_ID='@transcriptorbot:example.com'
MATRIX_BASE_URL='https://matrix.server.com'
MATRIX_TOKEN='syt_xyz'
```
Run the bot
- `python transcriptorbot.py`
